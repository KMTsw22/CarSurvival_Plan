# Car Survivor x Boss Hunt — 핵심 기획서

## 컨셉
**뱀서 (몹 학살 + 자동 공격 + 레벨업) + 보스 헌팅 (거대 포탑 사냥)**
기존 뱀서는 "버티기"가 목적. 이 게임은 **"사냥하러 가기"**가 목적.

| 조작 | WASD 이동만 (공격 자동) |
|------|------|
| **승리** | **맵의 포탑 전부 파괴 → 다음 챕터** |
| **패배** | **차량 HP 0 = 게임 오버 (뱀서 원본)** |

---

## 핵심 루프
```
시작: 맵에 포탑 5~6개 배치
    ↓
초반: 중앙에서 뱀서 플레이 (몹 학살 + 경험치 + 레벨업)
    ↓
중반: 강해지면 가까운 거대 포탑부터 돌격
      접근 시 경고 연출 + 몹 밀도 폭증 → 보스전 느낌
      포탑 공격 회피하면서 데미지 넣기
    ↓
포탑 HP 0 → 대폭발 연출 + 보상 + 구역 안전화 + 맵 전체 난이도 감소
    ↓
반복 → 마지막 포탑 파괴 = 챕터 클리어
```

### 포탑이 살아있을수록 불리한 구조
- 포탑 살아있음 → 웨이브 점점 강해짐
- 포탑 하나 부술 때마다 → 맵 전체 난이도 감소
- **빨리 부수면 쉬워지고, 늦으면 감당 불가** → 돌격 타이밍 판단이 핵심

## 뱀서 유지 요소

| 요소 | 유지 |
|------|------|
| 몬스터 100% 나한테 몰려옴 | O |
| 자동 공격 | O |
| 대량 학살 쾌감 | O |
| 레벨업 → 무기/패시브 선택 | O |
| 차량 죽으면 바로 게임 오버 | O |
| 10~15분 한 판 | O |

### 뱀서와 다른 점 (차별화)
- 버티는 게 아니라 **포탑을 부수러 가야 함**
- 포탑 근처는 **위험 구역** (몹 밀도 + 디버프)
- 포탑 파괴 순서 = **매 판 다른 전략**
- **언제 돌격할지** 타이밍 판단 = 뱀서에 없는 새로운 축

## 거대 포탑 (5종) — 보스급 존재감

맵에서 멀리서도 보이는 거대 구조물. 포탑 하나 부수는 게 **보스 하나 잡는 느낌**.

### 포탑 접근 시 연출
```
포탑 접근 → 경고 사이렌 + 화면 흔들림 + BGM 전환
    ↓
포탑 공격 범위 진입 → 회피하면서 데미지 넣기
    ↓
동시에 주변 몹 밀도 폭증 → 몹 처리 + 포탑 공격 동시 진행
    ↓
포탑 HP 0 → 대폭발 연출 + 보상 + 구역 안전화
```
> 전용 체력바 (보스 체력바처럼 화면 상단 표시)

### 포탑 목록

| 포탑 | 외형 | 공격 패턴 | 파괴 효과 |
|------|------|----------|----------|
| 드릴 타워 | 땅을 뚫고 솟은 거대 드릴 | 지면 균열 → 범위 데미지 | 지형 붕괴, 몹 경로 변화 |
| 레일건 타워 | 거대 포신, 조준 레이저 | 일직선 고데미지 빔 (예고선 표시) | 폭발 → 주변 몹 전멸 |
| 실드 제너레이터 | 거대 구체, 에너지 돔 | 돔 안 몬스터 무적 + 플레이어 디버프 | 돔 소멸, 몹 약화 |
| 하이브 타워 | 벌집형 거대 구조물 | 몬스터 무한 스폰 | 스폰 중단, 웨이브 급감 |
| 재밍 타워 | 거대 안테나, 전파 방출 | 무기 오작동 + 시야 교란 | 시야 해제, 무기 정상화 |

> **어떤 포탑부터 부술지** = 매 판 핵심 전략

## 성장 시스템

### 인게임 (레벨업)
몬스터 처치 → 경험치 → 레벨업 → **3개 중 1개 선택** (뱀서 방식)

| 카테고리 | 예시 |
|---------|------|
| 무기 강화 | 머신건, 화염방사기, 레이저 등 (8종, 자동 발사) |
| 패시브 | SpeedUp, DamageUp, HpUp 등 (7종) |

> 진화 시스템: 머신건(Lv5) + 오일슬릭 → 화염 머신건 등

### 메타 성장 (런 밖)
파츠 가챠 (골드) → 파츠 업그레이드 (골드+나사) → 영구 스탯 강화
등급: 일반/희귀/에픽/전설 (최대 Lv10, +50%)

### 희귀 드랍 (0.1%)
| 아이템 | 효과 |
|--------|------|
| 🧲 자석 | 맵 전체 경험치 즉시 흡수 |
| ⛽ 연료 | 주유소 생성, 10초간 HP 회복 |

|--------|------|------|
