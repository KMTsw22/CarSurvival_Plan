# Car Survivor — 테이블 설계서 (v1.0)

> 총 11개 테이블 | ID 형식: `{접두어}_{3자리 숫자}` | 참조는 반드시 ID로

---

## 1. TB_Car (차량)

| 컬럼명 | 타입 | 설명 | 예시 |
|--------|------|------|------|
| car_id | string (PK) | 차량 고유 ID | `CAR_001` |
| car_name | string | 차량 이름 | 스포츠카 |
| car_type | enum | SportsCar / SUV / Truck | SportsCar |
| base_hp | float | 기본 체력 | 100 |
| base_speed | float | 기본 이동속도 (units/s) | 5 |
| base_atk_speed | float | 기본 공격속도 (shots/s) | 1.5 |
| base_damage | float | 기본 공격 데미지 | 15 |
| collision_damage | float | 충돌 데미지 | 10 |
| passive_desc | string | 고유 패시브 설명 | 공격 쿨타임 -15% |
| passive_type | enum | 패시브 종류 (CooldownReduce / DefenseUp / CollisionReflect) | CooldownReduce |
| passive_value | float | 패시브 수치 | 15 |
| unlock_cost | int | 해금 비용 (골드) | 0 |
| unlock_currency_id | string (FK) | → TB_Currency.currency_id | `CUR_001` |
| unlocked_by_default | bool | 기본 해금 여부 | true |
| sprite_key | string | 스프라이트 리소스 키 | `spr_car_sports` |

## 7. TB_Wave (웨이브 타임라인)

| 컬럼명 | 타입 | 설명 | 예시 |
|--------|------|------|------|
| wave_id | string (PK) | 웨이브 고유 ID | `WAVE_000` |
| time_min | int | 시간 (분) | 0 |
| phase_name | string | 구간명 | 튜토리얼 |
| mon_id | string (FK) | → TB_Monster.mon_id | `MON_001` |
| spawn_per_30s | int | 30초당 스폰 수 | 4 |
| max_enemies | int | 동시 최대 적 수 | 15 |
| difficulty_scale | float | 난이도 배율 | 1.0 |
| note | string | 특이사항 | 적응 구간 |

**참조 관계:** `mon_id` → `TB_Monster`

> 같은 시간대에 여러 적이 스폰되면 **행이 여러 개** 생김 (예: 2분대에 러스티 세단 + 스파이크 바이크 = 2행)

## 8. TB_Level (레벨 테이블)

| 컬럼명 | 타입 | 설명 | 예시 |
|--------|------|------|------|
| level | int (PK) | 레벨 | 1 |
| required_exp | int | 필요 경험치 (누적) | 30 |
| required_exp_gap | int | 필요 경험치 (구간) | 30 |
| difficulty_multiplier | float | 웨이브 난이도 배율 | 1.0 |
| grade_common_pct | int | 일반 등급 파츠 등장 확률 (%) | 70 |
| grade_rare_pct | int | 레어 등급 파츠 등장 확률 (%) | 25 |
| grade_epic_pct | int | 에픽 등급 파츠 등장 확률 (%) | 5 |
| grade_legendary_pct | int | 레전더리 등급 파츠 등장 확률 (%) | 0 |
| note | string | 비고 | 초반 튜토리얼 구간 |

## 9. TB_Reward (보상 테이블)

| 컬럼명 | 타입 | 설명 | 예시 |
|--------|------|------|------|
| reward_id | string (PK) | 보상 고유 ID | `RWD_001` |
| reward_trigger | enum | 보상 발동 조건 (RunComplete / BossClear / AdWatch / DailyLogin) | RunComplete |
| currency_id | string (FK) | → TB_Currency.currency_id | `CUR_001` |
| amount | int | 지급 수량 | 100 |
| bonus_multiplier | float | 보너스 배율 (광고 등) | 1.0 |
| note | string | 비고 | 런 완료 기본 골드 |

**참조 관계:** `currency_id` → `TB_Currency`

> 한 트리거에 여러 재화를 주려면 행을 나눔 (예: 런 완료 시 골드 100 + 스크랩 20 = 2행)

## 11. TB_Shop (상점)

| 컬럼명 | 타입 | 설명 | 예시 |
|--------|------|------|------|
| shop_id | string (PK) | 상품 고유 ID | `SHOP_001` |
| shop_name | string | 상품 이름 | 스타터 패키지 |
| shop_type | enum | IAP / InGame / Ad | IAP |
| price_type | enum | 결제 수단 (RealMoney / Gold / Scrap) | RealMoney |
| price_amount | int | 가격 | 1900 |
| price_currency_id | string (FK, nullable) | → TB_Currency.currency_id (인게임 재화일 때) | null |
| reward_type | enum | 보상 종류 (Currency / Part / Car / Cosmetic) | Currency |
| reward_target_id | string (FK) | → 해당 테이블의 ID | `CUR_001` |
| reward_amount | int | 보상 수량 | 1000 |
| is_one_time | bool | 1회 구매 제한 여부 | true |
| release_phase | enum | MVP / v1.5 / v2 | MVP |
| note | string | 비고 | 첫 구매 유도용 |

**참조 관계:** `price_currency_id` → `TB_Currency`, `reward_target_id` → `TB_Car` / `TB_Part` / `TB_Currency` (reward_type에 따라 다름)

## 테이블 관계도 요약

```
TB_Currency ←──── TB_Car.unlock_currency_id
    ↑
    ├──── TB_Reward.currency_id
    ├──── TB_Shop.price_currency_id
    └──── TB_Shop.reward_target_id (reward_type=Currency일 때)

TB_Part ←──── TB_Evolution.material_a_id
    ↑    ←──── TB_Evolution.material_b_id
    │    ←──── TB_Evolution.result_part_id
    ├──── TB_MonsterDrop.special_drop_id
    └──── TB_Shop.reward_target_id (reward_type=Part일 때)

TB_Monster ←──── TB_MonsterDrop.mon_id
    ↑
    └──── TB_Wave.mon_id

TB_PartGrade ←── TB_Level (등급 확률 참조 기준)

TB_Car ←──── TB_Shop.reward_target_id (reward_type=Car일 때)
```

## 데이터 무결성 체크리스트

- [ ] TB_Evolution의 material_a_id, material_b_id, result_part_id 모두 TB_Part에 존재하는가
- [ ] TB_Wave의 mon_id가 TB_Monster에 존재하는가
- [ ] TB_MonsterDrop의 mon_id가 TB_Monster에 존재하는가
- [ ] TB_Level의 등급 확률 합계가 100%인가
- [ ] TB_Part의 drop_weight 합계가 의미있는 값인가
- [ ] TB_Shop의 reward_target_id가 reward_type에 맞는 테이블에 존재하는가
- [ ] TB_Reward의 currency_id가 TB_Currency에 존재하는가
- [ ] TB_Car.unlock_currency_id가 TB_Currency에 존재하는가
