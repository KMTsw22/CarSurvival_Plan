# Car Survivor x Race — 핵심 기획서

## 컨셉
**뱀서 (자동 공격 + 레벨업) + 레이싱 돌파**
몹을 피하는 게 아니라 **뚫고 전진한다.** 멈추면 죽는다.

| 조작 | WASD 이동 (공격 자동) |
|------|------|
| **승리** | **결승선 도달 (or 랩 완주)** |
| **패배** | **차량 HP 0 = 게임 오버** |

---

## 핵심 루프
```
출발
    ↓
도로 위 몹이 사방에서 몰려옴 → 자동 공격으로 학살하며 전진
    ↓
경험치 수집 → 레벨업 → 무기/패시브 선택 (뱀서)
    ↓
구간마다 거대 포탑이 길을 막음 → 부수면서 돌파
    ↓
속도 유지 = 생존. 느려지면 몹에 둘러싸여 죽음
    ↓
결승선 도달 = 챕터 클리어
```

### 핵심: 속도 = 생존
- 빠르게 달리면 → 몹이 따라오지 못함, 안전
- 느려지면 → 몹에 포위, HP 급감
- 포탑 구간에서 강제 감속 → 가장 위험한 순간
- **"언제 밟고, 언제 피할지"가 매 순간의 판단**

---

## 레이싱 요소

| 요소 | 적용 |
|------|------|
| 부스트/니트로 | 몹 밀집 구간 돌파, 포탑 구간 이탈용 |
| 드리프트 | 커브 구간 + 드리프트 중 공격 범위 확대 |
| 장애물 | 바리케이드, 오일 웅덩이 = 감속 위험 |
| 갈림길 | 루트 선택 → 쉽지만 보상 적음 vs 어렵지만 보상 많음 |

## 뱀서 유지 요소

| 요소 | 유지 |
|------|------|
| 몹이 나한테 몰려옴 | O |
| 자동 공격 | O |
| 대량 학살 쾌감 | O |
| 레벨업 → 무기/패시브 선택 | O |
| 차량 죽으면 게임 오버 | O |

### 뱀서와 다른 점 (차별화)
- 버티기가 아니라 **뚫고 전진**
- **속도 관리**가 생존의 핵심
- 포탑이 길을 막음 → **보스 헌팅 + 레이싱 장애물**
- 갈림길 → **루트 선택 전략**
- 멈추면 죽음 → **공격적 플레이 강제**

## 거대 포탑 (5종) — 길을 막는 보스급 장애물

트랙 중간중간 배치. 포탑 구간 진입 시 강제 감속 + 전투.

### 포탑 접근 시 연출
```
포탑 구간 진입 → 경고 사이렌 + BGM 전환
    ↓
강제 감속 → 몹 밀도 폭증 + 포탑 공격 시작
    ↓
포탑 회피하면서 데미지 넣기
    ↓
포탑 파괴 → 대폭발 + 도로 개방 + 속도 회복
```
> 전용 체력바 (보스 체력바 스타일)

### 포탑 목록

| 포탑 | 외형 | 공격 패턴 | 파괴 효과 |
|------|------|----------|----------|
| 드릴 타워 | 도로를 막는 거대 드릴 | 지면 균열 → 범위 데미지 | 도로 개방 + 경험치 대량 |
| 레일건 타워 | 거대 포신, 조준 레이저 | 일직선 고데미지 빔 (예고선) | 폭발 → 주변 몹 전멸 |
| 실드 제너레이터 | 도로 위 에너지 장벽 | 통과 불가 + 몹 무적화 | 장벽 소멸, 통과 가능 |
| 하이브 타워 | 도로변 벌집 구조물 | 몹 무한 스폰 | 스폰 중단 |
| 재밍 타워 | 거대 안테나 | 무기 오작동 + 시야 교란 | 무기 정상화 |

## 트랙 구조

### 한 챕터 = 하나의 트랙

```
출발선 ─── 구간1 ─── [포탑] ─── 구간2 ─── [갈림길] ─── 구간3 ─── [포탑] ─── 결승선
              │                    │            ↙↘          │
           몹 소수            몹 대량     쉬운길/어려운길   몹 폭증
```

### 구간별 구성

| 구간 | 내용 |
|------|------|
| 초반 구간 | 몹 소수, 레벨업 집중, 속도 익히기 |
| 중반 구간 | 몹 대량 + 첫 포탑 등장, 갈림길 선택 |
| 후반 구간 | 몹 폭증 + 포탑 연속, 속도 관리 극한 |
| 결승선 | 도달 = 클리어 |

### 갈림길 선택

| 루트 | 특징 |
|------|------|
| 고속도로 (쉬움) | 넓은 길, 몹 적음, 보상 적음 |
| 골목길 (어려움) | 좁은 길, 몹 밀집, 보상 많음 + 희귀 아이템 |
| 포탑 루트 (보스) | 추가 포탑 등장, 파괴 시 대량 보상 |

## 성장 시스템

### 인게임 (레벨업 — 뱀서 방식)
몬스터 처치 → 경험치 → 레벨업 → **3개 중 1개 선택**

| 카테고리 | 예시 |
|---------|------|
| 무기 강화 | 머신건, 화염방사기, 레이저 등 (8종, 자동 발사) |
| 패시브 | SpeedUp, DamageUp, HpUp 등 (7종) |

> 진화 시스템: 머신건(Lv5) + 오일슬릭 → 화염 머신건 등

### 메타 성장 (런 밖)
파츠 가챠 (골드) → 파츠 업그레이드 (골드+나사) → 영구 스탯 강화
등급: 일반/희귀/에픽/전설 (최대 Lv10, +50%)

### 희귀 드랍 (0.1%)
| 아이템 | 효과 |
|--------|------|
| 🧲 자석 | 맵 전체 경험치 즉시 흡수 |
| ⛽ 연료 | 주유소 생성, 10초간 HP 회복 |
| 🚀 니트로 | 5초간 무적 + 최대 속도 돌진 |
